# WordPress Article Skill

この skill は WordPress の無料記事を作成するためのものです。

## 目的
- 無料記事として有料級の価値を持つ記事を作る
- 「思考を取り戻す仕事術」の軸で信頼を積み上げる
- 有料noteへの自然な導線を設計する

## 参照ファイル
作業前に必ず以下を確認すること。
- README.md
- CLAUDE.md
- workflow.md
- templates/wordpress-article.md
- criteria/wordpress-criteria.md
- templates/review-template.md

必要に応じて以下も参照すること。
- strategy/roadmap.md
- strategy/metrics.md
- strategy/links.md

## 最初に確認すること
ユーザーに以下を確認する。
1. 記事タイトル
2. 想定ペルソナ
3. 記事の役割（集客 / 信頼構築 / 導線）
4. 接続したい note 記事
5. 体験談を主にするか、構造化を主にするか

## 作業フロー
1. 構成案を作成する
2. templates/wordpress-article.md に沿って初稿を作成する
3. criteria/wordpress-criteria.md に沿って自己レビューする
4. 95点未満なら、改善点を優先順位順に出す
5. 必要なら修正版まで出力する
6. 完成した記事は `contents/01_WordPress/drafts/` に `.html` および `.md` ファイルとして保存する
7. 公開後は `strategy/links.md` の WordPress テーブルに公開日・タイトル・URLを追記する

## 出力ルール
- 抽象論で終わらせず、構造と再現性を示す
- 実体験は補強として使い、主軸は構造化した説明にする
- 「弱い断定」は避ける
- ペルソナが「これは自分のことだ」と思える具体性を入れる
- 有料noteへの導線は、売り込みではなく「続きが自然に必要になる形」で入れる

## 出力形式
必要に応じて以下を分けて出力する。
- 構成案
- 初稿
- レビュー結果
- 修正版
