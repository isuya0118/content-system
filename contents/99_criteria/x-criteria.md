# X ポストレビュー観点

## 最重要観点
- [ ] この投稿は、非フォロワーにも止まって読まれるか
- [ ] この投稿は、想定ペルソナに対して「自分のことだ」と感じさせられるか
- [ ] この投稿は、いいね / リプ / クリック のどれを狙う投稿か明確か
- [ ] この投稿は、無料記事または有料記事への導線として機能するか

---

## バズ・拡散チェック
- [ ] 1行目で感情・違和感・問題提起が刺さるか
- [ ] 最初の1〜2行で「続きを読みたい」と思わせられるか
- [ ] 読者の痛み・損失・遠回りが言語化されているか
- [ ] 「へえ」で終わらず、反応したくなる余白があるか
- [ ] 内容が弱すぎず、少し引っかかりがあるか
- [ ] 誰かに共有したくなる要素（あるある・違和感・学び）があるか

---

## ペルソナ適合チェック
- [ ] 想定ペルソナが日常で感じている詰まりや悩みに直結しているか
- [ ] 読者が「これは自分の話だ」と感じる具体性があるか
- [ ] 投稿の温度感が、今のあなたの立場（実績過剰でない）と合っているか
- [ ] 実績ではなく、観察・仮説・解像度で勝負できているか

---

## 反応設計チェック
- [ ] 「共感」か「問い」が入っているか
- [ ] リプライが発生する設計になっているか
- [ ] 一方的な説明で終わっていないか
- [ ] 正解が1つではない問い、または経験共有しやすい形になっているか
- [ ] コメントしやすい終わり方になっているか（例：どう思いますか？ / 同じ人いますか？）

---

## クリック導線チェック
- [ ] 記事リンクを貼る理由が明確か
- [ ] 読者にとって「今読むべき理由」があるか
- [ ] 無料記事 / 有料記事の価値ポイントが具体的に伝わっているか
- [ ] URLがただ置かれているだけでなく、クリックしたくなる文脈があるか
- [ ] 元ポストだけでなく、必要ならリプライでもクリック導線を補強できているか

---

## 種別チェック（価値提供 or 宣伝）

### 価値提供型
- [ ] 構造化された気づき or 実践ポイントが含まれているか
- [ ] 「自分もこれだ」と思える具体性があるか
- [ ] 無料で読んでも価値がある投稿になっているか
- [ ] 1投稿1テーマになっているか

### 宣伝型
- [ ] 記事の価値ポイントが具体的に示されているか
- [ ] 売り込みが前面に出ていないか
- [ ] 「読まないと損する理由」があるか
- [ ] 記事に飛んだ後の期待値とズレていないか
- [ ] 記事URLを記載する場合、クリックしたくなる文脈と一緒に置かれているか（必要に応じて）

---

## 運用方針チェック
- [ ] 価値提供 3：宣伝 1 の比率を維持しているか
- [ ] 今のアカウント規模に対して、宣伝が多すぎないか
- [ ] 同じテーマの焼き直し・横展開ができる投稿になっているか
- [ ] 固定ポストや過去ポストとの役割分担ができているか

---

## 文体チェック
- [ ] 短く、強く、分かりやすいか
- [ ] 誰にでも当てはまる抽象表現で終わっていないか
- [ ] 断定しすぎず、でも弱くもないか
- [ ] あなたのテーマ（思考を取り戻す仕事術）と一貫しているか
- [ ] 読者の感情（焦り・共感・悔しさ・納得）を動かせているか

---

## 無料記事導線チェック
- [ ] 無料記事に送る場合、記事の入り口として十分な問題提起になっているか
- [ ] 無料記事の価値が自然に伝わっているか
- [ ] 「まずは無料で読んでほしい理由」があるか
- [ ] 無料記事へ送る方が適切な投稿か、有料へ直送すべき投稿かが明確か

---

## 有料記事導線チェック
- [ ] 有料記事に送る場合、無料で解決しきれない課題設定になっているか
- [ ] 「買う理由」が投稿文だけで伝わるか
- [ ] 価格に対する納得感を邪魔しない設計か
- [ ] 今のフォロワー規模・信頼残高で、いきなり有料へ送っても不自然でないか

---

## NGチェック
- [ ] 実績なしに「成功」を語っていないか
- [ ] 抽象的で誰にでも当てはまる表現になっていないか
- [ ] 「思考を取り戻す仕事術」の軸からズレていないか
- [ ] 良いことを言っているだけで、行動や反応に繋がらない投稿になっていないか
- [ ] 宣伝色が強すぎて、非フォロワーに嫌われる構造になっていないか

---

## 最終判定
- [ ] この投稿は、今のアカウント規模でも十分に伸びる可能性があるか
- [ ] この投稿は、想定ペルソナから反応（いいね / リプ / クリック）を取れるか
- [ ] この投稿は、無料記事または有料記事への導線として機能すると言えるか
- [ ] 95点未満なら、公開せずに修正すべき箇所が明確になっているか
