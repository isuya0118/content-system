# WordPress 無料記事レビュー観点

## 最重要観点
- [ ] 想定ペルソナが「これは自分のための記事だ」と感じられるか
- [ ] 無料記事として読者に十分な価値を与えられているか（有料級の満足感があるか）
- [ ] 有料noteに進む理由が自然に生まれているか
- [ ] 「売り込み」ではなく「続きを読みたい」にできているか

---

## ペルソナ適合チェック
- [ ] 読者が抱えている具体的な悩み・詰まりポイントを言語化できているか
- [ ] 読者が「自分のことだ」と思える具体的な場面が描けているか
- [ ] タイトルと導入で、誰向けの記事かが分かるか
- [ ] 記事全体が想定ペルソナからズレていないか

---

## 構成チェック
- [ ] 導入で共感・問題提起・記事の目的が自然に提示されているか
- [ ] 問題提起（共感・あるある）があるか
- [ ] 原因が「構造」として説明されているか
- [ ] 気づきが抽象化されているか（「〜という構造になっている」形式）
- [ ] 再現性（実践方法）が提示されているか
- [ ] 「思考を取り戻す仕事術」テーマへ回収されているか
- [ ] 各章の役割が重複していないか
- [ ] 読後に「結局どうすればいいか」が残るか
- [ ] HTMLファイルにH1タグが含まれているか
- [ ] 関連するXポストが記事内に引用されているか（oEmbed用URLをそのまま段落に記載）
- [ ] 他のWordPress記事への内部リンクが、内容に応じた箇所に設置されているか

---

## 無料記事としての価値チェック
- [ ] 抽象論で終わっていないか（構造と再現性が提示されているか）
- [ ] 「気づき」だけでなく「実践の入口」まで提示できているか
- [ ] 読者が「無料なのにここまで分かるのか」と感じる内容か
- [ ] 有料記事を読まなくても、単体の記事として満足できるか
- [ ] それでもなお、有料記事を読みたくなる余白が残っているか
- [ ] 読者が明日から一つでも試せる内容になっているか

---

## 内容チェック
- [ ] 実体験は「補強」として使われているか（主軸になっていないか）
- [ ] 仮説ベースの内容は「観察・気づき」として提示されているか
- [ ] 読者が「自分のことだ」と思える具体性があるか
- [ ] 行動を促す記述があるか（読む理由が明確か）
- [ ] 記事の主張が一貫しているか
- [ ] 「良いことを言っている」で終わらず、読者の認識を変えられているか

---

## 有料note導線チェック
- [ ] 有料記事への導線が中盤と末尾に配置されているか
- [ ] 導線の位置が「興味が最大になる箇所」になっているか
- [ ] 導線文に「なぜ今読むべきか」が含まれているか
- [ ] 導線文が売り込みすぎず、記事の流れに自然に溶け込んでいるか
- [ ] 誘導先の有料noteが、その無料記事の続きとして自然か
- [ ] 「どちらの有料記事へ送るべきか」が明確に設計されているか
- [ ] 読者が導線を見て「これ欲しい」と思える具体性があるか
- [ ] 中盤と末尾の導線文言が異なっているか（同じ文言の繰り返しになっていないか）
- [ ] 導線文に価格（300円・500円等）を記載していないか

---

## 売れる導線になっているかのチェック
- [ ] 無料記事で問題意識を十分に高められているか
- [ ] 無料記事だけでは解決しきれない部分が明確か
- [ ] 有料記事で解決できることが、無料記事本文の流れの中で自然に伝わるか
- [ ] 「今読まなくていい理由」より「今読みたい理由」の方が強いか

---

## 文体チェック
- [ ] です・ます調に統一されているか（だ・である調が混在していないか）
- [ ] 「弱い断定」をしていないか（例：〜だと思います）
- [ ] 「強い仮説提示」になっているか（例：〜という構造になっています）
- [ ] 誰にでも当てはまる抽象表現になっていないか
- [ ] 抽象 → 具体 → 行動 の流れがあるか
- [ ] 読者の感情（焦り・共感・納得）に触れられているか

## 文字数チェック
- [ ] 本文の文字数が2,500〜3,500字の範囲に収まっているか
- [ ] 短すぎて価値が薄くなっていないか（2,500字未満は要加筆）
- [ ] 長すぎて離脱を招いていないか（3,500字超は要削除）

---

## NGチェック
- [ ] 実績なしに「成功」を語っていないか
- [ ] 自分語りが主役になっていないか
- [ ] 誰にでも当てはまる抽象的な表現になっていないか
- [ ] 説明だけで終わっていないか（行動に繋がっているか）
- [ ] 売り込みが前面に出ていないか
- [ ] 有料記事への導線が唐突・不自然になっていないか

---

## 最終判定
- [ ] このWordPress記事は、無料記事として有料級の価値があるか
- [ ] このWordPress記事は、想定ペルソナに対して十分に刺さるか
- [ ] このWordPress記事は、有料noteへの導線として機能すると言えるか
- [ ] 95点未満なら、公開せずに修正すべき箇所が明確になっているか
