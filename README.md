# content-system

AI時代に「分かったつもり」で止まらず、一次情報を根拠に考えられる力を言語化し、
ブログ・note・Xを通じて発信していくコンテンツ運用プロジェクト。

## テーマ
- 思考を取り戻す仕事術
- 一次情報の読み方
- AI活用と根拠ある判断
- 本業と副業をつなげる思考

## 対象読者
- 忙しい会社員
- 主にIT・SE職
- AIは使っているが、説明やレビューで「それどこ情報？」と詰まりやすい人
- 本業の経験を副業や発信に活かしたい人

## メディアごとの役割
- WordPress：無料記事を蓄積し、SEO資産として育てる
- note：有料記事を販売し、収益化を担う
- X：認知拡大・流入獲得・反応検証を行う

## 公開先
- WordPress: https://isuya0118.com/
- note: https://note.com/seminomise
- X: https://x.com/isuya0118

## コンテンツ方針
- 抽象論で終わらせず、構造と再現性を示す
- 実体験は補強として使い、主軸は構造化した説明にする
- 無料記事は「気づき」と「一部の再現性」まで出し、有料記事で体系化する
- 売り込みすぎず、自然な導線で有料記事へつなげる

## 目的
- 無料記事・有料記事・X投稿を一貫したテーマで制作する
- 「思考を取り戻す仕事術」という軸で信頼を積み上げる
- 将来的にWordPress・note・Xを連動させた収益基盤を作る
